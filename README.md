# Vue Datatables

> Server Side Processing Datatables written with Vuejs Framework. 

Reference: https://datatables.net/examples/styling/bootstrap4

## Build

Seed dummy data
```sh
./datatables seed --records 150
```

Run demo server
```sh
./datatables run demo-server
```
